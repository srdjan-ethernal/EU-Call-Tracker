# AID4SME Application Checklist

## 1. Eligibility

- [ ] Choose lead company: Serbia / Malta / Slovenia.
- [ ] Confirm lead is SME or startup under EU SME definition.
- [ ] Confirm lead did not receive AID4SME Open Call #1 funding.
- [ ] Confirm lead submits only one AID4SME Open Call #2 application.
- [ ] Confirm country eligibility for the selected lead company.
- [ ] Decide single applicant or consortium up to 2 entities.
- [ ] If consortium: confirm partner role and whether partner requests funding.

## 2. Challenge

- [ ] Select exactly one challenge code.
- [ ] Write one-sentence fit to challenge.
- [ ] List available product, prototype, dataset, model or demo.
- [ ] Identify industrial playground/use-case dependency.
- [ ] Define expected TRL now and target TRL after 14 months.

## 3. Proposal

- [ ] Use mandatory Annex 2 proposal template.
- [ ] Keep within formatting and page limits.
- [ ] Proposal summary under 2,000 characters.
- [ ] Challenge alignment answer under 5,000 characters in F6S.
- [ ] Address Novelty/Innovation, Impact, Implementation and Team explicitly.
- [ ] Define SMART KPIs.
- [ ] Explain Green Deal and resource optimisation pathway.
- [ ] Explain IPR, data access and exploitation.

## 4. Budget

- [ ] Total requested contribution <= EUR 200,000.
- [ ] Requested contribution = 70% of total eligible project costs.
- [ ] No subcontracting included.
- [ ] Equipment costs use depreciation only.
- [ ] Personnel costs mapped to work packages.
- [ ] Travel/subsistence and purchase costs justified.
- [ ] Indirect costs calculated according to guide.

## 5. Documents

- [ ] Company registration extract.
- [ ] SME declaration evidence.
- [ ] Financial statements or management accounts.
- [ ] Ownership / UBO information.
- [ ] VAT/tax number and bank details.
- [ ] Team CVs.
- [ ] Relevant references, pilots, demos or customer letters.

## 6. Submission

- [ ] F6S company profile complete.
- [ ] All F6S mandatory fields complete.
- [ ] Annex 2 proposal uploaded as PDF.
- [ ] Terms accepted.
- [ ] Declaration of Honour accepted for contracting stage.
- [ ] SME Declaration accepted for contracting stage.
- [ ] Submit before 15 July 2026, 17:00 CEST.
- [ ] Save proof of submission.
